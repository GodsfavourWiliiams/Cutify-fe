# CUTIFY WEB FE

This repo house the frontend for CUTIFY website (test)

### Getting Started

1. Clone the repo
2. Install dependencies
3. Run the app

### Prerequisites

- Node.js
- Yarn

### Project setup

1. Clone the repo

```sh
git clone [repoLink]
```

2. Install dependencies

```sh
yarn
```

3. Compile and Hot-Reload for Development

```sh
yarn dev
```

4. Type-Check, Compile and Minify for Production

```sh
yarn build
```

5. Lint with ESLint

```sh
yarn lint
